# Dongnan Hu Homepage

This repository is now a lightweight static GitHub Pages homepage.

## Structure

- `index.html`: the homepage content.
- `assets/css/style.css`: page layout and responsive styling.
- `assets/js/main.js`: mobile navigation and active-section highlighting.
- `images/graduated_photo.png`: portrait image used in the Home section.
- `images/research-uav-navigation.png`: selected publication and project thumbnail.
- `images/research-3d-reconstruction.png`: research direction thumbnail.
- `images/research-aerial-manipulator.png`: research direction thumbnail.
- `assets/DongnanHu_CV.pdf`: CV link target.
- `files/Motion_Planning_and_Control_of_Hybrid_Flying_Crawling_Quadrotors.pdf`: paper PDF link target.

GitHub Pages can serve this directly from the repository root. No Jekyll, Ruby, or AcademicPages build is required.
